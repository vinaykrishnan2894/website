PokerBoss

Baazi Games · Senior Product Manager

A club-based real-money poker ecosystem built from scratch: mobile client + real-time multiplayer backend + back-office control plane.

The Problem

The real challenge in poker isn't the game — it's the business operating system. PokerBoss was built to be that OS: trustworthy, configurable, scalable infrastructure for clubs, agents, and operators to run private poker communities.

Key Problems → Solutions

⚡

Real-time Reliability

Server-authoritative state management, reconnect logic, and session recovery — so a dropped connection never means a lost hand or disputed pot.

⚖️

Dispute Resolution

Comprehensive hand history logs with action timelines and admin decision tooling — giving support teams everything they need to resolve disputes fairly and fast.

🏦

Chip Economy Control

Designed chip sources/sinks, transfer limits, role-based adjustments, and abuse prevention — keeping the economy honest and operators in control.

🔐

Back-office Permissions

Role-based access control across support, risk, finance, and super-admin tiers — right data for the right person, nothing more.

🎛️

Configurable Tables

Game toggles (Bomb Pot, VPIP tracking, CallTime) with guardrails that prevent broken game states — operators self-serve, devs stay out of the loop.

Impact

Shipped a full multiplayer poker ecosystem from 0 → soft launch

Admin portal reduced manual ops load across support, risk, and finance teams

Server-authoritative validation improved table stability and fairness

Configurable club controls enabled operators to self-serve game management
